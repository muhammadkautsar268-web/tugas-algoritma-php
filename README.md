## Judul Program

Program Menghitung Nilai Rata-Rata Mahasiswa

---

## Deskripsi

Program ini dibuat menggunakan bahasa pemrograman PHP untuk menghitung nilai rata-rata mahasiswa berdasarkan data yang disimpan dalam array. Program juga menentukan grade setiap mahasiswa menggunakan percabangan (if-elseif-else).

---

## Fitur

- Menampilkan data mahasiswa.
- Menggunakan array untuk menyimpan nilai.
- Menggunakan fungsi untuk menghitung rata-rata.
- Menentukan grade berdasarkan nilai rata-rata.
- Menampilkan hasil dalam tampilan web yang rapi.

---

## Teknologi yang Digunakan

- PHP
- HTML
- CSS
- XAMPP
- Visual Studio Code

---

## Cara Menjalankan Program

1. Install XAMPP.
2. Aktifkan Apache.
3. Simpan folder **tugas-algoritma-php** ke dalam folder **htdocs**.
4. Buka browser.
5. Ketik alamat berikut:

```
localhost/tugas-algoritma-php/
```
